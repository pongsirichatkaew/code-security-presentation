# API08 — Security Misconfiguration

<div class="mt-8 text-lg opacity-90">

Security issues don’t always come from bad code.  
Sometimes the system is exposed because of how it’s <span v-mark.orange="1">configured</span>.

</div>

<ul class="mt-8 space-y-4 text-lg">
  <li>Debug features left enabled</li>
  <li>Default credentials or open endpoints</li>
  <li>Secrets or internal info exposed by mistake</li>
</ul>

<p class="mt-8 opacity-80">
Nothing crashes. The system runs normally — but attackers learn too much.
</p>
