<p class="mb-4 opacity-80">
This endpoint looks simple, but it makes a dangerous assumption.
</p>

```ts
app.get('/api/v1/users/:id', (req, res) => {
  // Finds a user by ID from the request
  const user = db.findUserById(req.params.id);

  // Returns the user data directly
  res.json(user);
});
```
<ul class="mt-6 space-y-3 text-lg">
   <li v-click> 
      <span v-mark.orange="1">findUserById(id)</span> returns a user <span class="opacity-70">(no context of who is calling)</span> 
   </li>
   <li v-click> The API trusts the <code>:id</code> from the request </li>
   <li v-click> There is <strong>no ownership check</strong> between <code>req.user</code> and the returned user object </li>
</ul>
